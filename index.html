Everything in one bot

One bot to run the whole room.

Music, moderation, rewards, and a browser dashboard — no juggling five different bots and half a dozen tabs.

Music that keeps up

Clean, low-lag playback with queues, autoplay, and true 24/7 mode. Powered by Lavalink so it stays smooth even on a busy server.

Web dashboard

Control every server from the browser. Everything updates live over a socket — change the queue or a setting and it lands instantly.

Composer

Build embeds and announcements visually — colors, fields, buttons, the lot — and send them without touching a single line of code.

34 achievements

A full unlock system that rewards the people who actually keep your server alive. Wired through one place, so nothing double-fires.

Moderation

Lock a channel down in one command and open it back up just as fast. Escalation and per-server rules keep things calm without babysitting.

Config sync

Export your whole setup to a file and import it somewhere else. Set a server up once, then bring the same config anywhere you like.

The control room

Your server, live in the browser.

Open the dashboard and you're looking at the real state of the bot — not a snapshot. Skip a track, tweak a setting, or lock a channel, and everyone sees it happen at once.

  • Live everything. The queue, listeners, and settings stream in real time.
  • Per-section access. Hand out just the parts of the dashboard each person should touch.
  • Every server at once. Run all the servers Harmonix is in from a single window.
